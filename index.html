When did it rain last?

Enter a U.S. ZIP Code or paste coordinates from a maps link to see how long it has been since measurable rain and how much rain has fallen in the last 7, 14, and 21 days.

These figures come from a global weather analysis dataset—not a rain gauge next to your home. Treat them as a broad estimate for the location you pick: helpful for the big picture, but not the same as measuring rain in your backyard.
